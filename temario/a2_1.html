A2.1 Fundamentos de la red

Pregunta guía

¿Qué principios permiten que múltiples dispositivos compartan información?

Introducción

Una red permite comunicar dispositivos para compartir datos y servicios.

Explicación detallada

Las redes se clasifican por alcance y propósito: LAN, WAN, PAN y VPN. Internet es una red de redes, no un sistema único centralizado.

La dirección IP identifica lógicamente cada dispositivo para enrutar datos entre origen y destino.

Conceptos clave

  • Red informática
  • Tipos de red
  • Internet
  • Dirección IP

Ejemplo práctico

Una escuela usa LAN interna y salida a Internet; el profesorado accede de forma segura mediante VPN fuera del centro.

Aplicaciones en el mundo real

E-learning, servicios cloud, colaboración remota.

Errores frecuentes de los estudiantes

Confundir Internet con Wi‑Fi local; creer que IP es siempre estática.

Conexiones con otros temas del curso

A2.3 (transmisión), A2.4 (seguridad), A1.3 (interfaces de red).

Resumen

  • Las redes conectan dispositivos y recursos.
  • Internet interconecta millones de redes.
  • IP es clave para identificar y comunicar nodos.
